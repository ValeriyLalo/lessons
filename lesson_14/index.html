Практична форма

Тип text (за замовченням) - Однорядковий ввід даних

Тип email - Однорядковий ввід поштової адреси

Тип hidden - Приховане поле. Використовується разом з value

Тип tel - поле вводу телефонного номеру

Тип password - поле вводу паролю

Тип number - поле вводу числа

Тип color - поле вибору кольору

Тип search - поле пошуку / disabled

Тип date - поле вибору дати

Обрати містa

Тип radio - одиночний вибір. Для однієї групи радіо кнопок має бути одинакове значення атрибуту name. Варто вказувати значення атрибуту value.

Виберіть марку авто

Тип file - додати файл (файли). Атрибут multiple - додавання групи файлів. Атрибут accept - фільтрує формати файлів.

textarea - ввід багаторядкового тексту

Тег select - випадаючій список. Атрибут multiple. Тег option - елемент списку. Атрибут selected - вибраний за замовченням.

button - Кнопка
type="button" - кнопка нічого не робить
type="reset" - обнуляє (скидає) поля вводу у формі. Має знаходитить в середені тегу form.
type="submit" - (за замовченням) відправляє форму. Має знаходитить в середені тегу form.
